# pixeer-examples · nexora dashboard

A fictional fintech dashboard with a **⌘K AI spotlight** — type a command, watch [Pixeer](https://github.com/Heiscel/pixeer) execute it live in the UI.

**Stack:** React 19 · Vite · Tailwind CSS · Framer Motion  
**AI:** `google/gemma-3-27b-it` via [OpenRouter](https://openrouter.ai)

## Getting started

```bash
pnpm install
pnpm dev
```

Open the app, click **Ask Pixeer** (or press `⌘K`), add your OpenRouter API key in the ⚙ panel, then try:

- *"Add a task called Fix the login bug"*
- *"Go to the Tasks tab and create a new task"*
- *"Mark the first task as complete"*

## What is Pixeer?

[Pixeer](https://github.com/Heiscel/pixeer) is a 9KB in-browser AI agent runtime. It gives AI agents eyes and hands inside your real browser tab — no headless browsers, no scraping, no extensions.
